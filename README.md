# 🛒 SmartCart Customer Analytics

A simple Python script to clean the SmartCart customer dataset and engineering new meaningful features.

## 🚀 Key Features

* **Data Cleaning:** Handles missing data in the `Income` column by filling it with the median.
* **Age Calculation:** Derives the customer's exact age from their birth year.
* **Total Spending:** Combines spending across various product categories into a single `Total_Spending` metric.
* **Total Children:** Combines the number of kids and teenagers to get the total household child count.

## 🛠️ Built With

* **Python** (Pandas, Matplotlib, Seaborn)
